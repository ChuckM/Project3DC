This directory holds the project files and source code for building the library, 
tests, and demos under Win9x and Visual C++ 5.0 (SP1). Directories are as follows:
	src - subdirectories for the library, include file, demos, and tests.
	lib - 3rd party libraries (png and zlib)
	include - 3rd party includes (png and zlib)
	bin - Pre-buit demo stuff.
